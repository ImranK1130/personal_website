Project 2File Compression

{ C++ }

This project is a file compression program that utilizes the Huffman algorithm to compress and decompress files or strings. The program begins by reading in the input file or string and creating a frequency map of the characters it contains. This frequency map is used to build an encoding tree, which is a data structure that represents the relationships between the characters in the input and their corresponding encoded values.

Once the encoding tree has been constructed, the program uses it to generate an encoding map, which maps each character in the input to its corresponding encoded value. This encoding map is then used to compress the input, resulting in a compressed version of the file or string. A decoding map is also generated, which allows for the decompression of the compressed input.

The program includes several functions to facilitate these processes. The "buildFrequencyMap" function reads in the input file or string and creates the frequency map, while the "buildEncodingTree" function uses the frequency map to construct the encoding tree. The "buildEncodingMap" and "buildDecodingMap" functions use the encoding tree to generate the encoding and decoding maps, respectively. The "encodeData" function uses the encoding map to compress the input, and the "decodeData" function uses the decoding map to decompress the input.

In addition to these core functions, the program also includes a "freeTree" function that frees the memory allocated for the Huffman tree, as well as a "compare" class that is used to prioritize the nodes in the encoding tree based on their character frequencies. The program also makes use of data structures such as hashmaps and priority queues, as well as file reading and classes. Overall, this code provides a solid foundation for a file compression program that utilizes the Huffman algorithm.
